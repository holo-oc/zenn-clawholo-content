「なんとなく」投稿していては伸びない。数字を見て、次に何を改善するかを決める必要がある。

この記事では、SNS投稿の分析の仕方と、データから導くアクションを紹介する。

## 分析する指標とその意味

### 到達系指標

| 指標 | 意味 | 重要度 |
|------|------|--------|
| インプレッション | 表示回数 | ⭐⭐⭐ |
| リーチ | ユニーク表示人数 | ⭐⭐⭐ |
| フォロワー数 | フォローしてくれている人 | ⭐⭐ |

**ポイント：** インプレッションがリーチの3倍以上なら、同じ人に複数回見られている（好ましい）。

### エンゲージ系指標

| 指標 | 意味 | 目安（率） |
|------|------|-----------|
| いいね率 | 共感の度合い | 3-5% |
| コメント率 | 対話の度合い | 0.5-1% |
| 保存率 | 価値の高さ | 1-2% |
| シェア率 | 拡散力 | 0.1-0.5% |

**計算式：**
```
いいね率 = いいね数 ÷ リーチ × 100
保存率 = 保存数 ÷ リーチ × 100
```

### コンバージョン系指標

| 指標 | 意味 |
|------|------|
| プロフィール訪問 | 興味を持った人 |
| リンククリック | 詳細を知りたい人 |
| フォロー | 継続的に見たい人 |
| 問い合わせ/DM | 購入検討者 |

## 分析の手順

### Step 1：上位3投稿を選ぶ

過去30日間で、以下の基準で選ぶ：
- インプレッションが高い
- エンゲージ率が高い
- 保存数が多い

### Step 2：共通点を探す

**チェック項目：**
- [ ] 投稿タイプ（動画/画像/カルーセル）
- [ ] トピック（どんな内容）
- [ ] 投稿時間
- [ ] キャプションの長さ
- [ ] ハッシュタグ
- [ ] 最初の1文（フック）

### Step 3：下位3投稿も分析

伸びなかった投稿の共通点も探す。

**よくある失敗：**
- 話題がマイナーすぎる
- 文字だけで視覚的でない
- 投稿時間が悪い
- フックが弱い

## データから導くアクション

### ケース1：インプレッションが低い

**原因：** アルゴリズムに拾われていない

**対策：**
- トレンド音源/トレンドトピックを使う
- ハッシュタグを見直す
- 投稿時間を変える
- エンゲージを促す仕掛けを入れる

### ケース2：インプレッションは高いがエンゲージが低い

**原因：** 興味はあるが、価値を感じていない

**対策：**
- 内容を具体化する
- 共感ポイントを増やす
- キャプションを改善
- CTA（行動喚起）を明確に

### ケース3：エンゲージは高いがフォローに繋がらない

**原因：** 投稿は良いが、プロフィールに魅力がない

**対策：**
- プロフィール文を見直す
- 固定投稿を設定
- ストーリーズで人柄を見せる

### ケース4：保存数が多い

**良い兆候：** 価値の高い内容

**対策：**
- 同じテーマの続編を作る
- より詳細な版を作る
- シリーズ化する

## 分析の頻度

| 頻度 | やること |
|------|---------|
| 毎日 | 前日の投稿を簡易チェック |
| 毎週 | 週間レポート作成、傾向確認 |
| 毎月 | 月次分析、戦略見直し |
| 四半期 | 長期トレンド分析、目標修正 |

## まとめ

分析の目的は「数字を見る」ことではなく「次の行動を決める」こと。伸びた投稿のパターンを増やし、伸びなかったパターンを減らす。

毎週30分、数字と向き合う習慣をつけよう。
