副業を始めたいけど、何からやればいいかわからない。そんな人が陥る落とし穴は「準備の段階で疲れて終わる」こと。

この記事では、最短で結果を出すために最初にやるべき5つのステップを紹介する。

## 1. 稼げるスキルを1つに絞る

「デザインもできるし、動画もできるし、文章も書ける」は売れない。

**なぜか：**
- クライアントは「何でもできる人」より「この作業が得意な人」を探す
- 単価が上がらない
- 勉強が終わらない

### 絞り方

**まずは「今できて、人に価値を提供できること」から**

例：
- Excelで資料整理が得意 → データ入力代行
- Instagramの運用を会社でやってる → SNS運用代行
- 文章を書くのが好き → ブログ記事執筆

**チェック：**
- [ ] 1時間で◯◯できる
- [ ] 人から「それやってくれる？」と言われたことがある
- [ ] 楽にできる（苦労しない）

## 2. ポートフォリオを作る

実績がなくても、練習作品でOK。

### 作り方

**ステップ1：架空のクライアントを想定**
- 「渋谷のカフェ」
- 「フィットネスジム」
- 「個人ブランディングコーチ」

**ステップ2：実際に作る**
- デザインなら架空のロゴ/フライヤー
- 文章なら架空のブログ記事
- 動画なら架空のプロモーション動画

**ステップ3：課題と解決策を書く**
```
【課題】
新規顧客の獲得

【解決策】
SNSで若年層にアピールできるビジュアル制作

【成果】
（架空でもOK）SNSフォロワー30%増加
```

## 3. 案件を取れる場所に登録する

副業マッチングサイトに登録して、実績を積む。

### おすすめプラットフォーム

| サイト | 特徴 | 向いている人 |
|--------|------|-------------|
| クラウドワークス | 案件数が多い | 初心者〜 |
| ココナラ | スキル販売型 | 具体的なサービスがある人 |
| ランサーズ | 単価が高め | 中級者〜 |
| Twitter | 直接契約 | 発信力がある人 |

### 最初の提案の書き方

**ダメな例：**
「よろしくお願いします。がんばります。」

**良い例：**
```
{クライアント名}様

御社の◯◯について拝見しました。
△△の点で改善の余地があると感じ、ご提案させていただきます。

【提案内容】
- ○○
- △△

【なぜ私が適任か】
- 過去に□□の実績（具体的数字）
- ◇◇の強み

まずはお試しで1件からでもご検討ください。
```

## 4. 単価を上げる基準を決める

最初は安くてもいい。でも「いつ値上げするか」を決めておく。

### 目安

| 実績 | 目標単価 |
|------|---------|
| 0-3件 | 時給換算1,000円でもOK（実績作り） |
| 4-10件 | 時給換算2,000円 |
| 11-20件 | 時給換算3,000円 |
| 21件〜 | 時給換算5,000円以上 |

### 値上げのタイミング

- 常に予約が埋まる状態が続く
- クライアントから「もっとやってほしい」と言われる
- 同じクオリティでもっと安い人が見つからない

## 5. リピーターを作る仕組み

1回きりの仕事ではなく、継続的な関係を作る。

### 方法

**納品時に次を提案する**
```
今回の◯◯、お役に立てましたでしょうか。

もしよろしければ、毎月△△を継続的にご支援させていただけませんか。
（月額○万円）
```

**定期的なフォロー**
- 納品1週間後：「使い心地はいかがですか？」
- 1ヶ月後：「追加で必要なものはありますか？」

**特典を作る**
- 継続契約で1ヶ月無料
- まとめて3ヶ月契約で10%オフ

## 月5万円の計算

| パターン | 単価 | 件数/月 |
|---------|------|--------|
| A | 5,000円 | 10件 |
| B | 10,000円 | 5件 |
| C | 25,000円 | 2件 |
| D | 50,000円 | 1件（継続） |

最初はAから始めて、段階的にDを目指す。

## まとめ

副業で稼ぐコツは「小さく始めて、早く実績を作り、段階的にスケールする」こと。

完璧な準備は不要。今週中にスキルを1つ選んで、来週には最初の提案を出す。それだけで多くの人と差がつく。
