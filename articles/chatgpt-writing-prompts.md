## はじめに

ChatGPTを使って文章を書いているけれど、「なんかAIっぽい」「自分の言葉にならない」と感じていませんか？問題はツールではなく、プロンプトの与え方にあります。この記事では、実務で使える具体的なプロンプトを10個紹介します。

## プロンプトの基本構造

効果的な文章改善プロンプトは、以下の4要素で構成されます：

1. **役割（Role）**：どんな専門家として振る舞うか
2. **文脈（Context）**：誰に向けた文章か、どんな場面か
3. **指示（Instruction）**：具体的に何をしてほしいか
4. **制約（Constraint）**：文字数、トーン、避けるべき表現など

この構造を覚えておけば、あとは目的に合わせて中身を変えるだけです。

## 実務で使えるプロンプト10選

### 【敬語調整】

```
あなたはビジネスライティングの専門家です。
以下の文章を、取引先の部長宛てのメールとして適切な敬語に修正してください。
堅すぎず、適度に親しみやすいトーンでお願いします。

[対象の文章]
```

### 【簡潔化】

```
あなたは編集者です。
以下の文章を、元の意味を保ったまま3分の1の文字数に圧縮してください。
冗長な表現と重複している内容を削除し、核心的なメッセージだけを残してください。

[対象の文章]
```

### 【口語調への変換】

```
あなたはブロガーです。
以下のマニュアル的な文章を、20代後半の社会人が友達に話すような親しみやすい口調に変換してください。
「です・ます」は「だ・である」に変え、専門用語には必ず説明を加えてください。

[対象の文章]
```

### 【導入の強化】

```
あなたはマーケティングライターです。
以下の文章の冒頭を、読者の興味を引くように書き換えてください。
「共感できる悩み」を提示し、その解決策として本文につなげる構成にしてください。
文字数は100文字程度でお願いします。

[対象の文章]
```

### 【具体例の追加】

```
あなたは技術ライターです。
以下の抽象的な説明に、実際の数字や具体的なシナリオを含む例を3つ追加してください。
読者が「自分ごと」として捉えられるような、身近な例を選んでください。

[対象の文章]
```

### 【論理構成の整理】

```
あなたは論文指導のプロです。
以下の文章の論理構成を分析し、問題がある箇所を指摘してください。
その上で、以下の構造で再構成したバージョンを作成してください：
1. 主張（何が言いたいか）
2. 根拠（なぜそう言えるか）
3. 具体例（実際のケースは）
4. 結論（読者に何をしてほしいか）

[対象の文章]
```

### 【誤字脱字チェック】

```
あなたは校正のプロです。
以下の文章を校閲し、誤字脱字、文法ミス、不自然な表現をすべて指摘してください。
問題点ごとに、修正前→修正後の対比表形式で出力してください。

[対象の文章]
```

### 【要約作成】

```
あなたは要約の専門家です。
以下の文章を、3行の要約にまとめてください。
- 1行目：結論（何が言いたいか）
- 2行目：根拠（主な理由やデータ）
- 3行目：示唆（読者にとっての意味）

[対象の文章]
```

### 【トーン調整】

```
あなたはライティングコーチです。
以下の文章を、「親しみやすく、でも適度にプロフェッショナル」なトーンに書き換えてください。
絵文字は適度に使用し、固い表現は避けてください。

[対象の文章]
```

### 【CTA（行動喚起）の強化】

```
あなたはマーケターです。
以下の文章の最後に、読者が次に取るべき行動を明確に促す一文を追加してください。
「ぜひ試してみてください」ではなく、具体的な第一歩を示す表現にしてください。

[対象の文章]
```

## 使いこなすコツ

### 一度に一つの指示に絞る

「敬語に直して、かつ短くして、かつ具体例を追加して」という複数指示は、品質が落ちます。ステップバイステップで依頼するのが鉄則です。

### 出力後にフィードバックを与える

「もう少しカジュアルに」「逆に堅めに」といった修正指示は、同じスレッド内で続けて伝えられます。初回で完璧を求めず、対話的にブラッシュアップしてください。

### 良い出力は保存して再利用する

気に入った書き換えパターンができたら、そのプロンプトをテンプレートとして保存しましょう。同じような依頼が増えてきたら、それがあなた専用の最適化プロンプトです。

## まとめ

ChatGPTは「何でもできる魔法の道具」ではなく、「適切な指示を出せば高い成果を出す助手」です。プロンプトの構造を理解し、目的に応じて使い分けることで、文章の質は劇的に向上します。

今日から試せるのは：
1. 役割・文脈・指示・制約の4要素を意識する
2. 複数の指示は分けて依頼する
3. 気に入ったパターンはテンプレート化する

この3つを実践すれば、AIっぽい文章から卒業し、あなたらしい言葉を作る強みに変えられます。
