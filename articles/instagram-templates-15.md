Instagramで投稿しても反応がない。そんな時、大抵の人は「内容が悪い」と思う。でも実際は「構成」が問題なことが多い。

実際に集客できてるアカウントを分析したら、決まったパターンがあった。それを15個まとめた。

---

## テンプレート1：実績訴求型

```
【結論】[期間]で[成果]を出しました

【ビフォー】
・[悩み1]
・[悩み2]
・[悩み3]

【アフター】
・[改善1]
・[改善2]
・[改善3]

【方法】[具体的な方法]を実践しただけ

【募集】同じ成果を出したい人は
「[キーワード]」とコメントしてね
```

例：
```
【結論】3ヶ月で月商50万円達成しました

【ビフォー】
・投稿しても反応ゼロ
・何を書けばいいか分からない
・フォロワーが増えない

【アフター】
・毎日3件のお問い合わせ
・フォロワー3,000人突破
・リピート客が増えた

【方法】集客投稿テンプレートを使っただけ

【募集】同じ成果を出したい人は「テンプレート」とコメントしてね
```

## テンプレート2：ノウハウ提供型

```
【今日のテーマ】[テーマ]

【結論から言うと】[結論]

【理由】
① [理由1]
② [理由2]
③ [理由3]

【実践方法】
Step1: [ステップ1]
Step2: [ステップ2]
Step3: [ステップ3]

【保存してね】
この投稿は保存して何度も見返してください
```

---

## テンプレート3：共感・共鳴型

```
【わかる人いる？】[悩み/あるある]

【実は】[解決策/真実]

【私の場合】[自分の経験]

【あなたも】[行動喚起]

【コメントしてね】[同じ経験がある人]はコメント欄で教えて
```

---

## テンプレート4：限定・緊急型

```
【お知らせ】[限定情報]

【内容】[詳細]

【期間】[期間]まで

【対象】[対象者]

【応募方法】
① [手順1]
② [手順2]
③ [手順3]

【DM待ってます】興味ある人はDMで「[キーワード]」と送ってね
```

---

## テンプレート5：質問・調査型

```
【質問】[質問内容]

【選択肢】
A. [選択肢A]
B. [選択肢B]
C. [選択肢C]

【コメントしてね】A/B/Cどれ？コメント欄で教えて

【結果発表】[期間]後に結果を投稿します
```

---

## 使い方

1. 目的に合ったテンプレートを選ぶ
2. [ ]を自分の内容に置き換える
3. 投稿

月〜金でテンプレートをローテーションすると、考える時間が減って継続しやすくなる。

---

個人的には「実績訴求型」が一番問い合わせに繋がった。ただし、実際に実績がないとすぐバレるので注意。
