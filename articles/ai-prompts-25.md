ChatGPTで「何を聞けばいいか分からない」って人が多い。プロンプト（指示文）の書き方を変えるだけで、回答の質が全然違う。

実際に業務で使ってるプロンプトを25個まとめた。コピペして[ ]を埋めるだけ。

---

## 文章作成・ライティング（5個）

**1. ブログ記事作成**
```
あなたは[業界]の専門家です。

以下のトピックについて、初心者向けブログ記事を作成：
- トピック：[トピック]
- ターゲット：[対象読者]
- 文字数：[字数]程度
- トーン：[カジュアル/フォーマル]

構成：
1. 導入（問題提起）
2. 本論（3つのポイント）
3. まとめ（次のアクション）

SEOキーワード「[キーワード]」を含めて。
```

**2. SNS投稿作成**
```
以下を[プラットフォーム]向けに変換：

元の内容：[元の文章]

条件：
- 文字数：[字数]以内
- トーン：[トーン]
- ハッシュタグ：[個数]個
```

**3. メール返信作成**
```
以下のメールへの返信を作成：

受信メール：[メール内容]

条件：
- トーン：[丁寧/カジュアル]
- 伝えたいこと：[ポイント]
- 文字数：[字数]程度
```

**4. 商品説明文作成**
```
以下の商品説明文を作成：

- 商品名：[名前]
- 価格：[価格]
- ターゲット：[対象]
- 特徴：[特徴1]、[特徴2]、[特徴3]

構成：
1. キャッチコピー
2. ペイン喚起
3. 解決策
4. メリット3つ
5. CTA
```

**5. キャッチコピー生成**
```
以下のキャッチコピーを[個数]個作成：

商品：[名前]
ターゲット：[対象]
強み：[強み]

条件：[字数]以内、[トーン]な雰囲気
```

- [字数]以内

## 使い方のコツ

1. **具体性** → [ ]を具体的に埋める
2. **文脈** → AIに役割を与える（「あなたは〇〇」）
3. **制約** → 文字数・トーン・構成を明示
